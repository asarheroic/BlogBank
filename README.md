# 📝 BlogBank

**BlogBank** is a full-featured blogging platform inspired by Medium. Users can create and explore articles with a clean UI, markdown or rich text support, and personalized feeds.

Built as a learning project for mastering full-stack web development using modern, scalable technologies.

---

## 🚀 Features

- 🖋️ Create, edit, and delete blog posts
- 🧑‍💻 User authentication with JWT
- 🧠 Personalized article feed
- 🧾 Markdown or rich text editor
- 📦 Backend APIs with type-safe validation (Zod)
- 💾 PostgreSQL database with Prisma ORM
- 🌐 Hosted on Cloudflare Workers
